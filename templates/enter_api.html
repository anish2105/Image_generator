Why HuggingFace API Token?

  • Hugging Face provides and supports a wide range of open-source tools and technologies that enable users to build, train, and deploy ML models.
  • As we are not using OPENAI, we need to out-source an open source, therefore models available on HuggingFace are accessed through API calls
  • Since we are making an API call to HuggingFace, we are going to need HuggingFace Token

How to get your HuggingFace key?

  • Navigate to this URL: https://huggingface.co/settings/tokens
  • Create a write token and copy it to your clipboard
  • Enter your token below
